Keyboard Backlight - Dell - Ubuntu

 Posted on May 20, 2018  |  1 minutes (87 words)

I have realised my Dell laptop keyborad backlight doesn’t light up in Ubuntu. So did some browsing and changed the config file. Set the brightness level to > 0 using below, you need to change the value using super user (or root). Open the config using nano editor. sudo nano /sys/devices/platform/dell-laptop/leds/dell\:\:kbd_backlight/brightness change the value 0, I have changed it to 2, then save the config. Also if required we can set the timeout of the keyboard backlight. [Read More]
